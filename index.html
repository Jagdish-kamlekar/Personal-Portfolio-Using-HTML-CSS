My Projects

Login & Registration Using Servlet , Jsp MVC

The project implements a user registration and login system using JSP, Servlets, and JavaBeans. The RegisterBean class serves as a POJO to hold user details, and RegisterDao handles database interactions for user registration. The RegisterServlet captures user input from the registration form (register.jsp) and uses RegisterDao to insert the user into the database. The login.jsp handles user login by checking credentials, setting a session attribute for the logged-in user The home.jsp checks for an active session (via email) and redirects to the login page if no session exists Upon login, the user’s email is displayed with navigation options. The pages are styled with basic CSS to create a user-friendly interface .
